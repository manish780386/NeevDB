Getting started

NeevDB is on PyPI. Install it with pip, import it, and you have a working database in three lines.

Installation

Requires Python 3.7 or newer. No other dependencies.

# Install from PyPI
pip install neevdb

# Verify
pip show neevdb

Connect to a database

Pass a file path to NeevDB(). The file is created automatically if it doesn't exist. All data is stored as human-readable JSON.

from neevdb import NeevDB

# Creates 'mydata.json' if it doesn't exist
db = NeevDB("mydata.json")

# Defaults to 'neevdb.json'
db = NeevDB()
💡 The JSON file is fully human-readable. Open it in any text editor to inspect or back up your data.

Tables

Tables are collections of records — like sheets in a spreadsheet. Create one before inserting data.

# Create a table
db.create_table("users")

# List all tables
db.list_tables()
# → ['users']

# Delete a table permanently
db.drop_table("users")

CRUD operations

The four core operations — Create, Read, Update, Delete — cover everything you need to manage data.

Insert

Add a new record to a table. Every record automatically gets _id and _created_at fields.

record = db.insert("users", {
    "name": "Alice",
    "age":  25,
    "city": "Mumbai"
})

# Returns the full record:
# { name: Alice, age: 25, _id: 1, _created_at: 2026-01-01T... }

Find

Read records using find_all() to get everything, or find() with a condition function to filter.

# All records
all_users = db.find_all("users")

# Records matching a condition
adults = db.find("users", lambda r: r["age"] > 18)

# Mumbai users only
local = db.find("users", lambda r: r["city"] == "Mumbai")

# Count records
total = db.count("users")  # → 5

Update

Update all records matching a condition. Pass new field values as a dictionary. Returns the count of updated rows.

updated = db.update(
    "users",
    lambda r: r["name"] == "Alice",   # condition
    {"city": "Delhi", "age": 26}       # new values
)
# updated → 1

Delete

Delete all records matching a condition. Returns the count of deleted rows.

deleted = db.delete(
    "users",
    lambda r: r["age"] < 18
)
# deleted → 1
⚠️ Delete is permanent. There is no undo. Double-check your condition before running.

Query engine

Run SQL-like query strings with db.query(). Combine SELECT, WHERE, ORDER BY, and LIMIT freely.

SELECT

Use SELECT * for all columns, or name specific columns separated by commas.

# All columns
db.query("SELECT * FROM users")

# Specific columns only
db.query("SELECT name, city FROM users")

# Combined with other clauses
db.query("SELECT name FROM users WHERE age > 18 ORDER BY name LIMIT 5")

WHERE

Filter records by a condition. Works with text fields and number fields.

db.query("SELECT * FROM users WHERE age > 18")
db.query("SELECT * FROM users WHERE city = Mumbai")
db.query("SELECT * FROM users WHERE age != 25")
db.query("SELECT * FROM users WHERE age >= 21")

ORDER BY

Sort results by any field. Default is ascending (A→Z, 0→9). Add DESC to reverse.

# Ascending (default)
db.query("SELECT * FROM users ORDER BY name")

# Descending — highest first
db.query("SELECT * FROM users ORDER BY age DESC")

LIMIT

Return only the first N records. Useful combined with ORDER BY to get top results.

# First 5 records
db.query("SELECT * FROM users LIMIT 5")

# Top 3 highest-priced products
db.query("SELECT name, price FROM products ORDER BY price DESC LIMIT 3")

CLI shell

Run queries from your terminal without writing any Python. Launch the shell with one command.

Starting the shell

# Default database (neevdb.json)
python cli.py

# Specific database file
python cli.py mydata.json

CLI commands

NeevDB > SHOW TABLES
NeevDB > CREATE TABLE users
NeevDB > INSERT INTO users name=Alice age=25 city=Mumbai
NeevDB > SELECT * FROM users WHERE age > 18
NeevDB > SELECT * FROM users ORDER BY age DESC LIMIT 5
NeevDB > DELETE FROM users WHERE name = Alice
NeevDB > DROP TABLE users
NeevDB > HELP
NeevDB > EXIT
ℹ️ The CLI supports the same WHERE operators and ORDER BY / LIMIT syntax as the Python query engine.
